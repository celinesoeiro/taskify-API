# Taskfy API

#### An easy way to manage your tasks

### What can I do using Taskfy?

- Create a task
- View all your tasks and individual tasks
- Update a task
- Remove a task
- Mark a task as done
- Import tasks from csv files
- Categorize a task 

## Steps

✔️ Create a server
✔️ Create a local DB
✔️ Create POST
✔️ Create GET all
- Create GET by id
- Create PUT
- Create DELETE
- Create PATCH
